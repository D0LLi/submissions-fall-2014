What We Did

We used Eric's old analysis project from second term sophomore year. He analyzed rankings of the popular game Osu.
The original project was form based. Since we did not learn how to use that through flask, we decided to leave that part out.
Had there been a form, the user would be able to select certain parts of the table and limit the amount of data visible.
We also had to convert the code to a format that satisfied flask commands.
Originally, the project involved printing several lines of html markup, especially for the table.
Now, we converted the function so that it creates a string of html markup that would be inserted into a template.
